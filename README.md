## 🚀 Featured AI/ML Projects

### 🤖 [Reddit Bot — AI Posting Agent](https://github.com/aabhathanki/Reddit-AI-Agent)
*An intelligent AI agent built to automate posting, community listening, and interactive engagement.*
*   **Technologies:** Python, LangChain, LangGraph, Reddit API
*   **Core Highlights:**
    *   Designed a multi-step task orchestration pipeline using LangGraph to guide agent decisions.
    *   Replaced previous Twitter integration with a highly modular Reddit workflow.
    *   Automated community detection and relevant content creation.

### 🩺 [MediBot — Healthcare Chatbot](https://github.com/aabhathanki/MediBot)
*A patient-centric conversational assistant designed to address medicine and prescription queries safely.*
*   **Technologies:** Python, LangChain, LLM, Retrieval-Augmented Generation (RAG)
*   **Core Highlights:**
    *   Leveraged RAG pipelines to fetch precise medical answers based on custom knowledge bases.
    *   Designed structured validation layers in LangChain to prevent model hallucinations.
    *   Built a highly responsive conversational UI to guide patient queries.

### 🔍 [Product Comparison AI Agent](https://github.com/aabhathanki/ai-agent)
*A smart reasoning agent that crawls the web to perform live product comparisons.*
*   **Technologies:** Python, LangChain, ChatGroq, Tavily API
*   **Core Highlights:**
    *   Integrated Tavily API to fetch real-time specifications, pricing, and reviews.
    *   Configured Groq LLMs for low-latency reasoning and summary extraction.
    *   Formulates comparative Markdown tables automatically based on user query.

### 📰 AI Research Paper Agent
*An automated workflow to discover, rank, and summarize the latest AI/ML literature.*
*   **Technologies:** n8n, LLM, ArXiv API, Email Automation
*   **Core Highlights:**
    *   Pulls new pre-prints daily from the ArXiv API based on specific machine learning keywords.
    *   Evaluates and ranks over 50+ papers per execution using customized prompt criteria.
    *   Delivers a beautifully formatted HTML newsletter containing top 5 summaries straight to email inbox.
